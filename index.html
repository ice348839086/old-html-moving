Old HTML Moving / TrojanUI PoC

Move an old internal page without rewriting its legacy logic.

This demo shows three views of the same idea: the original legacy page, the Shadow DOM-based modern takeover, and a broken control group that uses plain innerHTML injection.

Preview 1

Original Legacy UI

Shows the ugly ERP-style table layout, global SimSun font, brown text, and hostile CSS pollution.
Preview 2

TrojanUI Experiment

Mounts the modern dark-card interface inside Shadow DOM while routing intent back to legacy functions.
Preview 3

innerHTML Control Group

Uses direct DOM injection without Shadow DOM, so the modern UI is visibly contaminated by legacy CSS.

Recommended Test Flow

  1. Open Original Legacy UI to see the baseline.
  2. Open TrojanUI Experiment, click Query Stock, and confirm the legacy alert receives the modern input.
  3. Check the bottom status panel for legacy:response back-writing.
  4. Open innerHTML Control Group to compare the CSS pollution failure case.
